# geoiplookup
--------------------------------------------------------------------------------

Geo Localization from a Given IP Address

# Notice
--------------------------------------------------------------------------------

This script was tested and work well in Python 2.7.6

# Changelog
--------------------------------------------------------------------------------
* **28/06/2017 - v1**
First Stable Release

# How to use it
--------------------------------------------------------------------------------

Example:

```sh
$ geoiplookup 216.58.222.14

# Output
IP: 216.58.222.14
Hosted by: Google
City: Mountain View
Country: United States
```
